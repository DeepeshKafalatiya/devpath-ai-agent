# DevPath AI 🚀
### Autonomous Multi-Agent Career & Skill Mentor
**Submitted for Agentic Arena 2026 - Round 2**

DevPath AI is a production-grade autonomous multi-agent framework designed to bridge the structural skill gap for engineering students by replacing static learning charts with self-correcting dynamic roadmaps.

## 🤖 Core Agent Architecture
1. **Trend Intelligence Agent (TIA):** Continuously tracks global engineering benchmark standards.
2. **Profile Evaluation Agent (PEA):** Parses user software footprints and scans active repository source configurations.
3. **Dynamic Curriculum Orchestrator (DCO):** Evaluates computed differences and streams hyper-personalized micro-task nodes.

## ⚙️ Tech Stack & Frameworks
- **Core Engine:** Python, FastAPI
- **Agent Reasoning Layer:** Google Gemini Flash API Integration
- **Frontend Layer:** Unified Asynchronous HTML5/JavaScript Matrix Dashboard
